About This Project

 Shoply is a mock e-commerce website that specializes in grocery delivery. This project was completed as a requirement for a Web-based Data Management course I am currently taking at the University of Windsor.

 The objective of the project was to design an e-commerce website that consisted of 3 different web pages coded in HTML and that use the CSS framework w3.css. We were also instructed to include a JavaScript operation in one of our pages. The operation I included was a slideshow found on the “About” Page.




This Website was Created By Josh Brown using HTML, Bootstrap 3, CSS, JavaScript and PHP
