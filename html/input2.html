About This Project

 This project is designed to demonstrate simple use of a web form using HTML, PHP, and JavaScript. This assignment was a requirement for a Web-Based Data Management course I am currently taking at the University of Windsor.

 The objective of the project was to create an HTML form that upon submission would be able to:
a)Ensure all necessary fields are filled
b)respond with a message back, depending on whether or not an ‘Active’ checkbox option had been selected.

 In order to do this I wrote a JavaScript function that checked to see if the required fields were filled. If they were, the program would continue to process the action to our PHP function. If the field was not filled an alert would be triggered to the screen, indicated which field was not filled.
 Upon submission, the a PHP function would first check to see if the ‘Active’ option was selected. If it was not, if would take the user-inputed first and last name, and print a message such as:

Josh Brown is NOT Active.

 If the ‘Active’ box was selected it would output the following message with the user-inputted data:

Greetings! Josh Brown currently has the title Web Developer and can be reached by phone at 5555555555 or email at thisemail.@email.com




This Website was Created By Josh Brown using HTML, Bootstrap 3, CSS, JavaScript and PHP
