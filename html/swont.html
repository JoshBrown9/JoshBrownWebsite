About This Project

 The objective when I took over as Webmaster for the SWONT Optimist website was simple. Take an outdated, stale website, and create something that captures the SWONT Optimist brand. When I first began, I wanted to create something that would look as good on a PC as it did on mobile. With this in mind I decided to use the mobile-first CSS framework Bootstrap 3. Though work on the mobile site is still underway, I feel that once completed, we will have achieved our goal.

 The Website consists mostly of static web pages that display various information and files that are used by both members, and those interested in becoming members of Optimist. There is also a password protected webpage designed for storing various files not needed to be seen by non-members.

 The site is still currently undergoing updates, with new pages and features planned to be added in the coming months.




This Website was Created By Josh Brown using HTML, Bootstrap 3, CSS, JavaScript and PHP
